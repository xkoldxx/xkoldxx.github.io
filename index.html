What I Do

Network Infrastructure

Enterprise network design and deployment using Cisco, Palo Alto, and UniFi. VLAN segmentation, zero-trust architecture, and SD-WAN — built by an engineer who designs these at enterprise scale daily.

Cloud & Hybrid Infrastructure

Azure and GCP migrations that don't break what already works. Hybrid cloud architecture, Microsoft 365 integration, and self-hosted alternatives that can dramatically reduce your SaaS spend.

Cybersecurity

Palo Alto next-gen firewalls, Cisco ISE network access control, endpoint protection, and compliance-ready security assessments. Not a checklist audit — a real security posture from someone who configures these in production.

Managed IT

Proactive monitoring, automated patching via Ansible, and real support when something breaks. A full IT department — monitoring, updates, troubleshooting, planning — minus the six-figure salary.

Structured Cabling

Clean Cat6/Cat6A runs, properly terminated patch panels, and network closets you won't be embarrassed to open. Professional installation that meets TIA standards and is built to last.

Security Camera Systems

UniFi Protect and enterprise IP surveillance — site survey, camera placement, installation, and remote access. Your footage stored on-site on hardware you own, not on someone else's cloud with a monthly fee.
